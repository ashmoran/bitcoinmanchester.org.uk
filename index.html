Bitcoin Manchester

Bitcoin meetup in Manchester, UK

Download this project as a .zip file Download this project as a tar.gz file

What?

Bitcoin Manchester is a meetup for Bitcoin users, miners, buyers, sellers or anyone else otherwise interested in Bitcoin in the north-west.

Where?

Bitcoin Manchester's first meetup will be at the MadLab on Edge Street in the Northern Quarter.

When?

The first meeting will be on Monday, June 24th 2013, 7-9pm (roughly). Come along!

More info

Follow us on Twitter: @bitcoin_mcr

Introduce yourself on the mailing list: bitcoin-manchester (sign up here if you want to receive announcements via email or Google Groups).

Website etc is work in progress, please bear with us while we get all the content up.
